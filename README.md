# ASP.NET MVC 作品 - PhoneShop 手機商店

### 開發環境
* Visual Studio 2017
    
### 使用技術
* ASP.NET MVC
* Entity Framework – Code First
* C#
* HTML
* CSS
* Bootstrap
* JavaScript
* JQuery
* AJAX

    * 產品依照條件做顯示
    * 產品分頁
