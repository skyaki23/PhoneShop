# ASP.NET MVC 作品 - PhoneShop 手機商店

## 網站連結
* https://phoneshopmvc2020.azurewebsites.net/

### 開發環境
* Visual Studio 2017 & 2019
    
### 使用技術
* ASP.NET MVC
* Entity Framework – Code First
* C#
* HTML
* CSS
* Bootstrap
* JavaScript
* JQuery
* AJAX
* SQL Server

### 作品介紹
### 1.首頁
* 輪播圖顯示最新上傳產品
* 顯示產品資訊並帶有加入購物車按鈕
* 產品可依照條件(搜尋關鍵字、產品排序、品牌查詢、價錢)做顯示
* 產品分頁
